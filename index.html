LSA Calculator API

Welcome to the Local Services Ads Calculator API. This service helps estimate advertising budgets for various industries using Google's LSA platform.

Available Endpoints

  • /health - Check API status
  • /calculate-budget - Calculate LSA budget (available in local environment only)
  • /test - Access the calculator interface

Note: The full calculation feature requires a server environment with browser capabilities and is not available in the serverless deployment.
